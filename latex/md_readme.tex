This project implements a 2-\/axis Core\+XY air hockey robot designed as part of the ME507 embedded systems course. The robot plays against a human by tracking the puck with computer vision and responding with predictive paddle movements.

The documentation pages are at \href{https://kyleas.github.io/air_hockey_robot/}{\texttt{ https\+://kyleas.\+github.\+io/air\+\_\+hockey\+\_\+robot/}}

\hypertarget{md_readme_autotoc_md10}{}\doxysection{\texorpdfstring{Overview}{Overview}}\label{md_readme_autotoc_md10}
The system is divided into two main components\+:


\begin{DoxyItemize}
\item {\bfseries{Sensing \& Vision (Raspberry Pi)}} ~\newline
 Uses Open\+CV to detect the puck and paddle on the board using HSV filtering, contour detection, and predictive path planning.
\item {\bfseries{Motion Control (STM32)}} ~\newline
 A Core\+XY-\/style robot arm driven by two stepper motors, controlled by an STM32 microcontroller using custom C firmware.
\end{DoxyItemize}

Key features include\+:
\begin{DoxyItemize}
\item Limit switch-\/based auto-\/homing
\item UART command interface between Pi and STM32
\item Predictive reflection-\/based path planning
\item Core\+XY motion control with synchronized velocity planning
