\doxysubsection{Detailed Description}
Core\+XY stepper motor management structure. 

Manages dual stepper motors in Core\+XY configuration with automatic calibration, soft limits, position tracking, and coordinated movement. Provides millimeter-\/based positioning with automatic step conversion.

\begin{DoxyNote}{Note}
Core\+XY Kinematics\+:
\begin{DoxyItemize}
\item Motor A (motor\+\_\+x)\+: Controls X+Y movement (A = X + Y)
\item Motor B (motor\+\_\+y)\+: Controls X-\/Y movement (B = X -\/ Y)
\item Forward\+: X = (A + B)/2, Y = (A -\/ B)/2
\item Inverse\+: A = X + Y, B = X -\/ Y 
\end{DoxyItemize}
\end{DoxyNote}
